🎯 Pointed To-Do

A Point-Based Productivity Tracker

📌 Project Description

Pointed To-Do is a desktop productivity application built using Java ☕ and Swing 🖥️.

The application allows users to create tasks, assign point values (0–5), mark tasks as completed, and calculate their overall productivity percentage 📊.

This project was developed to strengthen understanding of Java fundamentals, GUI development, and logical problem-solving 🧠.

🚀 Features

➕ Add new tasks

🎯 Assign point values (0–5)

✅ Mark tasks as completed

📊 Calculate total task points

📈 Calculate completed task points

🏆 Display performance-based feedback

❌ Automatically close after showing results

🛠 Technologies Used

☕ Java

🖥️ Java Swing (GUI)

📦 ArrayList

🎧 Event Handling (ActionListener, KeyListener)

🧠 How It Works

Each task has a name 📝 and a point value 🎯.

Task points are stored in an ArrayList 📦.

When a task is checked ✅, its points are added to a completed list.

When the user clicks Done 🎬, the program:

Calculates total points

Calculates completed points

Computes completion percentage

Displays a final feedback message 🏆

▶ How to Run

1️⃣ Open the project in a Java IDE (IntelliJ, Eclipse, or NetBeans).
2️⃣ Compile and run Main.java.
3️⃣ Start adding tasks and track your productivity 🚀.
